## 내용 강조
+ 중요한 것은 **굵게** 표시,
+ 주의할 것은 *기울이기*
+ `코드/명령어`는 따로 표시하자.
    + 굵게할 때는 * 두개로 감싼다(**bold**)
    + 이텔릭체는 * 한개로 감싼다(*italic*)
    + 코드 강조는 baccktick \` 한개로 감싼다(`code`)
